# Create Flask Web App
1. Create virtualenv
   > virtualenv venv
   - venv is the folder name that store virtualenv

2. Start virtualenv
   > venv\Scripts\activate

3. Install Flask
   > pip install flask


**Before push to git**
- create file .gitignore
  + add venv/ to .gitignore
